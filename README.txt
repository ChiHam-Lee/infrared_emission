本系统是基于STM32F103VB做的一个简易的光照感应红外发送的控制器，日常生活中，可以用于灯光的控制。比如说在楼道中，在光线充足时，打开开关，红外发送光照充足的数据，此时灯管将不能打开，当光线不足时，红外发送光线不足的数据，就可以成功打开灯管，达到了节能的效果。而且本系统的可塑性很强，只要将判断光照感应强度，改为判断环境温度，就可以用来控制日常生活中空调或者风扇的使用。

本次用到了红外了红外发送跟接收模块，还用到了模/数转换器，光照传感器等。需要两块实验板实现，首先一台实验板需要将老师所下发的红外通信程序下载到实验板中，用于接收本实验所发的数据，另一台实验板则将本程序编译并下载到实验板中。我在程序中用了adc模数转换器，感应周围光照强度，并用了一个if判断语句，来对光照强度即ADC中通道14的值进行判断，当4小于100时，则向外通过红外发送模块来发送值为0的数据，当光照强度大于100时，则发送值为1的数据，达到区别环境光照强度的目的。
